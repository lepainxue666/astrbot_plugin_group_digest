# astrbot_plugin_group_digest
基于Astrbot与QQ的插件，用于将群聊里的消息进行过滤汇总定时汇报
# 群重要消息汇总（Group Digest）

一个用于 AstrBot 的群消息辅助插件，用于自动过滤群聊中的噪音信息，
记录重要消息（@你、@全体成员、关键词），并支持定时或主动查询汇总。

---

## 功能特性

- 自动监听群聊消息
- 识别并记录以下“重要消息”：
  - @你
  - @全体成员
  - 命中关键词的消息
- 支持每日定时私聊汇总
- 支持私聊指令主动查询最近的重要消息
- 不影响正常群聊发言

---

## 使用方式

### 1. 自动记录

插件启用后，会自动在后台记录重要消息，无需额外操作。

### 2. 主动查询

私聊机器人发送：最近重要消息；最近重要消息 3天。

#目前属于测试阶段，机器人只能处于一个群聊里。
